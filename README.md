# Tableau Sales & Customer Insights Dashboard

## Project Overview
This project focuses on analyzing sales performance and customer behavior using Tableau dashboards. 

The objective was to transform raw sales data into interactive dashboards that help stakeholders understand key business metrics and trends.

## Dashboards Created

### 1. Sales Performance Dashboard
This dashboard provides an overview of sales performance metrics and trends.

Key features:
- Year-over-year comparison of Sales, Profit, and Quantity
- Monthly sales trend analysis
- Product subcategory sales comparison
- Weekly sales and profit performance against average

### 2. Customer Insights Dashboard
This dashboard analyzes customer behavior and contribution.

Key features:
- Customer growth and order trends
- Distribution of customers based on number of orders
- Identification of high-value customers
- Top 10 customers ranked by profit

## Tools Used
- Tableau
- Data Visualization
- KPI Analysis
- Dashboard Design

## Key Business Insights
- Certain product subcategories contribute higher sales but lower profit margins.
- A small group of customers contributes a significant portion of total profit.
- Weekly trend analysis helps identify performance fluctuations during the year.
